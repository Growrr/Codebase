The Codebase off-canvas sidebar can be used indepentantly, or it can be used as a modification to the grid system.

If used as a modification of a grid system, then instead of setting the col-[size]--span-* of a sidebar column on your lowest grid setting (xs, sm, md or lg), use the class col-[size]--off-canvas. You can also specify whether the behavior is to the left or right.

You will also need to add a navicon for toggling the sidebar.

Example: see the sidebar on this page at different screen widths.

<div class="off-canvas-navicon off-canvas-navicon--right">
    <i class="fa fa-bars" aria-hidden="true"></i>
</div>

<div class="xs--off-canvas off-canvas--right">

    <!-- Off-canvas sidebar content -->

</div>
