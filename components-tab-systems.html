Tab Systems

The Cosebase tab system comes in three types (top, side and accordion). Their HTML is the same; only a modifier class on the containing DIV needs to be altered to swap from one tab type to another.

Within Codebase tab systems, each tab label and their respective tab card can be placed next to each other, in pairs.

A tab label's data-tabs="" refers to its respective card's id="".

HTML
<div class="tabs tabs--top">

  <div class="tabs__label" data-tabs="tabs-example-1">Tab Label 1</div>
  <div id="tabs-example-1" class="tabs__card">
    <div class="tabs__card-content">
      <p>Tab Content 1</p>
    </div>
  </div>

  <div class="tabs__label" data-tabs="tabs-example-2">Tab Label 2</div>
  <div id="tabs-example-2" class="tabs__card">
    <div class="tabs__card-content">
      <p>Tab Content 2</p>
    </div>
  </div>

  <!-- Other tabs ... -->

</div>

Mixins
// COMPONENT: TABS
// ===============
// File: codebase/scss/07_component-mixins/_tab-systems.scss

@mixin tabs {
  margin: $base-line-height-rem 0;
  @include clearfix;
}

@mixin tabs__label {
  z-index: 0;
  padding: $tabs-label-padding;
  color: $deco-tabs-dark-text;
  background: $deco-tabs-dark-background;
  &:hover {
    color: $base-text-color;
    background: $deco-tabs-dark-background-hover;
    cursor: pointer
  }
  &--is-front {
    z-index: 2;
    color: $base-text-color;
    background: $deco-tabs-background !important;
    box-shadow: none;
    cursor: default
  }
}

@mixin tabs__card {
  z-index: 1;
  clear: both;
  float: left;
  width: 100%;
  height: 0;
  //font-size: $base-font-size;
  overflow: hidden;
  opacity: 0;
  &-content {
    border: $deco-border-thin-solid;
    padding: $tabs-card-padding;
    background: $deco-tabs-background;
  }
  &--is-front {
    height: auto;
    opacity: 1;
  }
}

//@mixin tabs--top {} // Not needed

@mixin tabs--top__label {

  border-right: $deco-border-thin-solid;
  border-left: $deco-border-thin-solid;
  box-shadow: $deco-box-shadow-inset-bottom;
}

  @include tabs__card;
  margin-top: -1px;
}

  position: relative;
  overflow-x: hidden;

  border-bottom: $deco-border-thin-solid;
  border-left: $deco-border-thin-solid;
  margin-bottom: $deco-spacing-xs;
  box-shadow: $deco-box-shadow-inset-right;
}

@mixin tabs--left__card {
  @include tabs__card;
  position: absolute;
  top: 0;
  right: 0;

  }
}

@mixin tabs--accordion {
  position: relative;
}

@mixin tabs--accordion__label {
  @include tabs__label;
  border-top: $deco-border-thin-solid;
  border-right: $deco-border-thin-solid;
  border-left: $deco-border-thin-solid;
  box-shadow: $deco-box-shadow-inset-bottom;
  &--is-front {
    background: $deco-tabs-background;
  }
}

@mixin tabs--accordion__card {
  @include tabs__card;
  margin-top: 0 !important;
  &--is-front {
    margin-bottom: $deco-spacing-xs;
    &:last-of-type {
      margin-bottom: 0;
    }
  }
  &-content {
    box-shadow: $deco-box-shadow-inset-bottom;
  }
  &:last-child .tabs__card-content {
    box-shadow: none !important;
  }
}

    Tabs system appearance
  


        

          Simply swap the tabs modifier class to alter the appearance of the tab system:
        


        
<div class="tabs tabs--top"> ... </div>

<div class="tabs tabs--side"> ... </div>

<div class="tabs tabs--accordion"> ... </div>
