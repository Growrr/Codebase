Notes:

  1. Dropdown menu behavior only happens on window widths above the navbar media query breakpoint. When the nemubar is collapsed on smaller screens, the child lists are displayed as indented lists within the stacked menu.
  2. On window widths narrower than the navbar breakpoint (i.e. for collapsed navbars), all links are followable (including parent links).
  3. On screens at or wider than the navbar breakpoint (i.e. for navbars as navbars), child menus are hidden by default (using the dropdown utility). Clicking the parent link once will open the dropdown, and clicking the parent link a second time will follow the parent link. (Clicking anywhere outside the dropdown except the parent link will dismiss the dropdown.)

navbar styling has therefore only been done as far as child menus (i.e. not sub-sub menus, etc).
