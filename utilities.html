In Codebase, utilities are HTML/CSS patterns that can be used on different elements and in different components (applicability depends on what the utility is). These utilities have been developed as mixins and some of them have also been used to build classes of the same name.

Utility Description Mixins Classes
Combos Used to un-round rounded corners on buttons and form elements when used inline.
Dropdown Used to create menubar dropdowns (include the dropdown mixins) or dropdowns on any element (e.g. use dropdown classes to create dropdowns for buttons).
Forms Alternative ratio boxes and checkboxes using Font Awesome. See also combos (above) for styling inline form elements.
Images Mixins and classes that control alignment.
Reset overlay A transparent whole-page layer that is raised above the webpage during the operation of dropdowns, menubar operation on phones, etc., providing a "click outside the opened thing to dismiss it" option. (The overlay component is prepended to the <body> by codebase.js – so you don't need add it yourself.) ()
Typography Some useful mixins and classes for controling lead paragraphs, drop caps and text alignment. (Color can also be controlled by .deco-* classes; see above.)
