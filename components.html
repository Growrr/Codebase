Codebase components are common webpage HTML/CSS patterns. Some components are partly comprised of Codebase utilities. Some have added functionality from a small jQuery script (e.g. popover, slideshow, tabs).

Codebase components attempt to be "unopinionated" in styling, as much as possible (meaning that there is less for designers to need to override).

Only where necessary, some decorative styling has been built in – all of which can be easily modified by overriding the decoration variables at codebase/scss/02_default-variables/_decoration.scss. In addition, in order to properly display some of these components, a few deco-* decoration classes have been employed – all of which are entirely optional.
