Incorporate the Codebase semantic grid mixins into your own layout, using your own choice of class names.

The Codebase semantic grid mixins are named similar to the Codebase classic grid classes, except that the column numerals are used as in the example SCSS stylesheet below (this arrangement of mixins would give you the example layout above).

You can also control the media query breakpoints using the Codebase breakpoint mixins, as below. 

.your-grid-wrapper-class {
  @include grid;
  @include grid--gutter;
}

.your-article-column-class {
  @include column-semantic;
  @include breakpoint__medium--up {
    @include column--span(8);
  }
  @include breakpoint__large--up {
    @include column--push(4);
  }
}

.your-sidebar-column-class {
  @include column-semantic;
  @include breakpoint__medium--up {
    @include column--span(4);
  }
  @include breakpoint__large--up {
    @include column--pull(8);
  }
}

.your-nested-grid-wrapper-class {
  @include grid;
  @include grid--gutter;
  @include grid--nested;
  > div {
    @include column-semantic;
    @include breakpoint__small--up {
      @include column--span(4);
    }
  }
}

.your-centered-column-class {
  @include column-semantic;
  @include breakpoint__small--up {
    @include column--centered(7);
  }
}
Mixin Explanation
@mixin grid The container mixin. Sets the max width (from variable $grid-max-width), does centering by auto margin right and left.

If a containing wrapper doesn't have this grid mixin included, then a grid system will expand to whatever the width of the container is, without maxing out at $grid-max-width. E.g. it could expand to he BODY width.
@mixin grid--gutter Gives right and left padding to all immediate child block elements (i.e. columns).
@mixin grid--nested Gives negative right and left margin to the grid container equal to the gutter width, so that a nested grid is flush right and left inside their containing column.
@mixin column-semantic Float left. (Min height 1px.)
@mixin column--centered(X) No float. Instead, centering is accomplished by auto margin right and left. Column spanning is as calculated out of $grid-max-columns (default is 12).
@mixin column--span(X) Column spanning, as calculated out of $grid-max-columns (default is 12).
@mixin column--push(X) Column pushing to the right, as calculated out of $grid-max-columns (default is 12).
@mixin column--pull(X) Column pulling to the left, as calculated out of $grid-max-columns (default is 12).
@mixin column--offset(X) Column offsetting from the left, as calculated out of $grid-max-columns (default is 12).
