For consistency throughout Codebase, some $deco- variables have been defined. Where necessary, a few of these variables have been used in the various utilities and components.

These same variables have been incoporated into a set of simple .deco- decoration classes. These classes can be used to quickly add consistent styling to your HTML directly.

You can modify these $deco* variables and .deco- classes, and add to them, in accordance with the style guide or visual identity of your website project.
