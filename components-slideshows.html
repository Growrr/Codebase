Components:

Slideshows

An easy-setup, lightweight, responsive slideshow. Capable of handling images or text, the Codebase slideshow component is suitable for displaying simple slideshow presentations, step-by-step tutorials, etc.

Example:

Title for Slide 1

Plus some descriptive text.

Title for Slide 2

Plus some descriptive text.

Title for Slide 3

Plus some descriptive text.

Title for Slide 4

Plus some descriptive text.

Title for Slide 5

Plus some descriptive text.

Title for Slide 6

Plus some descriptive text.

HTML
<div class="slideshow">
  <div class="slideshow__inner">

    <div class="slideshow__item"> ... </div>

    <div class="slideshow__item"> ... </div>

    <div class="slideshow__item"> ... </div>

  </div>
</div>
Mixins
// COMPONENT MIXINS: SLIDESHOW
// ===========================
// File: codebase/scss/07_component-mixins/_slideshows.scss

 @mixin slideshow {
  position: relative;
  margin-bottom: $base-line-height-rem;
  font-size: 0;
  line-height: 0;
}

@mixin slideshow__inner {
  position: relative;
  overflow: hidden;
  white-space: nowrap; // This makes the `.slideshow__item`s stay in one row
  font-size: 0;
  line-height: 0;
}

@mixin slideshow__item {
  display: inline-block;
  width: 100%;
  vertical-align: top;
  padding: 0 ($base-font-size * 3) 0 ($base-font-size * 3);
  font-size: $base-font-size;
  line-height: $base-line-height;
}

// Slideshow prev/next arrows

@mixin slideshow__arrows {
  position: absolute;
  display: flex;
  align-items: center;
  z-index: 1;
  top: 0;
  bottom: $deco-spacing-lg;
  font-size: $base-font-size;
  line-height: $base-line-height;
  span {
    display: flex;
    align-items: center;
    justify-content: center;
    width: $base-font-size * 2;
    height: $base-font-size * 2;
    border: 2px solid $deco-charcoal;
    border-radius: 50%;
    background: $deco-body;
    opacity: 0.3;
    &:hover {
      cursor: pointer;
      opacity: 1;
    }
  }
}

@mixin slideshow__prev {
  @include slideshow__arrows;
  left: $deco-spacing-sm;
  span:before {
    display: block;
    position: relative;
    left: -2px;
    font-family: FontAwesome;
    content: '\f053';
  }
}

@mixin slideshow__next {
  @include slideshow__arrows;
  right: $deco-spacing-sm;
  span:before {
    display: block;
    position: relative;
    right: -2px;
    font-family: FontAwesome;
    content: '\f054';
  }
}

// Slideshow pager

@mixin slideshow__pager-wrap {
  position: relative;
  bottom: 0;
  width: 100%;
  text-align: center;
}

@mixin slideshow__pager {
  display: inline-block;
  padding: $deco-spacing-xs;
  font-size: $base-font-size;
  line-height: $base-line-height;
  opacity: 0.3;
  &:before {
    font-family: FontAwesome;
    content: '\f10c';
  }
  &.is-current:before {
    font-family: FontAwesome;
    content: '\f111';
  }
  &:hover {
    cursor: pointer;
    opacity: 1;
  }
}
Classes
// COMPONENT CLASS-BUILDER: SLIDESHOW
// ==================================
// File: codebase/scss/11_component-classes/_slideshow.scss

.slideshow {
  @include slideshow;
  &__inner {
    @include slideshow__inner;
  }
  &__item {
    @include slideshow__item;
  }
  &__prev {
    @include slideshow__prev;
  }
  &__next {
    @include slideshow__next;
  }
  &__pager-wrap {
    @include slideshow__pager-wrap;
  }
  &__pager {
    @include slideshow__pager;
  }
}
JS (jQuery)

            

          


        


        

          Notes:
        


        
    
          
  1. 
            The simple setup consists of two outer block elements (.slideshow and .slideshow__inner) The inner block serves as the 'slider trolley' containing all the .slideshow__item blocks.
          
    2. 
          
  2. 
            The Codebase slideshow adds its own prev/next arrows and pager dots. It calculates the number of dots by counting your slides, i.e. counting your .slideshow__item blocks.
          
    3. 
          
  3. 
            The Slideshow will remain the height of your tallest slide plus the pager height. (The height does not auto-adjust while changing between slides of different heights).
          
    4. 
          
  4. 
            Only one instance of the Codebase slidershow component can be used per webpage.
          
    5. 
          
  5. 
            The Codebase slideshow component is not capable of looping or auto-run.
          
    6. 
        


        

          If you are need a slideshow with more capabilities, I recommend Slick by Ken Wheeler.
