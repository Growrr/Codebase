Classless Base Styles:

Forms

Note:
Further form styling can be added using form utilities.

Form elements expand to 100% of their container width. Give them structure by wrapping them in grid columns.

There is no added margin or padding around form elements – so nothing you need to override. Add whatever margin or padding (or not) as suits your own design.

Input boxes:

Only the button form elements require a class in order to be styled. See buttons.

Base Styles
// DEFAULT STYLES: FORMS
// =====================
// File: codebase/scss/04_base-styles/_forms.scss

form {
  //@include clearfix;
}

label {
  display: inline-block;
  padding: 6px 0;
}

@mixin form__element {
  -webkit-appearance:none;
  position: relative;
  width: 100%;
  min-height: 36px;
  outline: 0 none;
  border: $deco-border-thin-solid;
  @if $form-input-rounded == true {
    border-radius: $form-input-border-radius;
  }
  padding: 5px;
  color: $base-text-color;
  &::-webkit-input-placeholder,
  &::-moz-placeholder,
  &:-ms-input-placeholder,
  &:-moz-placeholder,
  &:placeholder {
    color: $base-text-color-muted;
  }
  &:focus {
    border-color: $base-text-color-hyperlink;
  }
}

input,
select {
  @include form__element;
}

select {
  padding: 5px;
  color: $base-text-color-muted;
}

select:not([multiple]) {
  height: 36px;
}

input[type="radio"],
input[type="checkbox"] {
  display: inline-block;
  width: 20px;
  min-height: auto;
  margin: 0;
  padding: 0 10px;
  border: 0;
  line-height: 1;
}

input[type='radio'] {
  -webkit-appearance: radio;
}

input[type='checkbox'] {
  -webkit-appearance: checkbox;
}

textarea {
  @include form__element;
	vertical-align: top;
	overflow: auto;
}
