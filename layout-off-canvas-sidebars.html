Off-Canvas Sidebars

The Codebase off-canvas sidebar can be used indepentantly, or as a modification to the grid system.

If you use the off-canvas sidebar as a modification of a grid system, then instead of setting the col-[size]--span-* of a sidebar column on your lowest grid setting (xs, sm, md or lg), use the class col-[size]--off-canvas. You can also specify whether the off-canvas is to the left or right.

You will also need to add a navicon for toggling the sidebar.

Be sure to set the off-canvas navicon data-off-canvas="" to point to its respective off-canvas sidebar id="".

It is possible to have more than one off-canvas sidebar. But only one can be activated at a time; one must be closed before another can be opened.

Example: see the sidebar on this page at different screen widths.

HTML
<div class="off-canvas-navicon off-canvas-navicon--right" data-off-canvas="sidebar-1">
    <i id="sidebar-1" class="fa fa-bars" aria-hidden="true"></i>
</div>

<div class="xs--off-canvas off-canvas--right">

    <!-- Off-canvas sidebar content -->

</div>
Variables
// DEFAULT VARIABLES: OFF-CANVAS SIDEBAR
// =====================================
// File: codebase/scss/01_default-variables/_off-canvas-sidebar.scss

$off-canvas-width: 250px !default;
$off-canvas-background: $deco-white !default;
$off-canvas-transition: all 400ms ease !default;
$off-canvas-navicon-background: $deco-white !default;
$off-canvas-navicon-color: $base-text-color-hyperlink !default;
$off-canvas-navicon-color-hover: $base-text-color-hyperlink-hover !default;
Mixins
// LAYOUT MIXINS: OFF-CANVAS SIDEBAR
// =================================
// File: codebase/scss/05_layout-mixins/_off-canvas-sidebar.scss

// Off-canvas navicon

@mixin off-canvas-navicon {
  position: fixed;
  top: 0;
  z-index: $reset-overlay--z-index + 2;
  padding: $deco-spacing-xs $deco-spacing-sm;
  font-size: 24px;
  line-height: 34px;
  color: $off-canvas-navicon-color;
  background: $off-canvas-navicon-background;
  transition: $off-canvas-transition;
  transform: translate3d(0, 0, 0);
  &:hover {
    color: $off-canvas-navicon-color-hover;
    cursor: pointer;
  }
}

@mixin off-canvas-navicon--right {
  right: 0;
  &.off-canvas--is-open {
    right: $off-canvas-width;
  }
}

@mixin off-canvas-navicon--left {
  left: 0;
  &.off-canvas--is-open {
    left: $off-canvas-width;
  }
}

// Off-canvas sidebar
// The extra 10px hides the box-shadow while off-canvas

@mixin off-canvas-right {
  right: -($off-canvas-width + 10px);
  &.off-canvas--is-open {
    right: 0;
  }
}

@mixin off-canvas-left {
  left: -($off-canvas-width + 10px);
  transition: all 0.4s;
  &.off-canvas--is-open {
    left: 0;
  }
}

@mixin off-canvas {
  position: fixed;
  top: 0;
  bottom: 0;
  width: $off-canvas-width;
  max-height: 100vh;
  z-index: $reset-overlay--z-index + 1;
  overflow: auto;
  background: $off-canvas-background;
  box-shadow: $deco-box-shadow-2x;
  transition: $off-canvas-transition;
  transform: translate3d(0, 0, 0);
}
Classes
// LAYOUT CLASSES-BUILDER: OFF-CANVAS SIDEBAR
// ==========================================
// File: codebase/scss/09_layout-classes/_off-canvas-sidebar.scss

// Off-canvas navicon

.off-canvas-navicon {
  @include off-canvas-navicon;
  &--right {
    @include off-canvas-navicon--right;
  }
  &--left {
    @include off-canvas-navicon--left;
  }
}

// Off-canvas sidebar

.xs--off-canvas {
  @include break__xs--only {
    &.off-canvas--right {
      @include off-canvas-right;
    }
    &.off-canvas--left {
      @include off-canvas-left;
    }
    @include off-canvas;
  }
}

.sm--off-canvas {
  @include break__sm--down {
    &.off-canvas--right {
      @include off-canvas-right;
    }
    &.off-canvas--left {
      @include off-canvas-left;
    }
    @include off-canvas;
  }
}

.md--off-canvas {
  @include break__md--down {
    &.off-canvas--right {
      @include off-canvas-right;
    }
    &.off-canvas--left {
      @include off-canvas-left;
    }
    @include off-canvas;
  }
}

.lg--off-canvas {
  &.off-canvas--right {
    @include off-canvas-right;
  }
  &.off-canvas--left {
    @include off-canvas-left;
  }
  @include off-canvas;
}

// Compensation for use if navbar is position:fixed;
// (`.body--navbar-is-fixed` gets added by the jQuery script.)

.body--navbar-is-fixed {
  .off-canvas-navicon--right,
  .off-canvas--right,
  .off-canvas-navicon--left,
  .off-canvas--left {
    margin-top: $navbar-height;
  }
}
