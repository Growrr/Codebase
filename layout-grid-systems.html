Three flavours of grid have been provided (Classic, Semantic and Flexbox). These all have similar display behaviour:

  • 12 columns – you can change this simply by modifying the $grid-max-columns variable in your chosen grid flavour
  • Flexible – column width is set using max-width, not width
  • Responsive – media query breakpoint widths: 768px, 1024px and 1200px – you can change these in scss/base/_base-breakpoints.scss
  • Pushing/pulling (or in case of the flexbox grid, re-ordering)
  • Offsetting
  • Nesting
  • (single column) centering

In addition, visibility classes have been provided, for showing/hiding items at different breakpoints.

Note: while all three grids have been included in this Repo instance of Codebase, users are encouraged to @include only their grid flavour of choice.
