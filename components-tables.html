Add class="table" to tables. This adds the styles that you see below. codebase.js adds a wrapper around tables with this class, enabling responsive overflow (horizontal scrolling). If the table width is wider than the width allowed in the webpage layout, then an inner shadow is added around the table to indicate that there is more to the table than what is visible.
