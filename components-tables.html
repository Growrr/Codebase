Add class="cb-table" to tables. This adds the styles that you see below. codebase.js adds a wrapper around tables that have this class, enabling responsive overflow (horizontal scrolling). If the table width is wider than the width allowed in the webpage layout, then a thick border is added to the right of the table to indicate that there is more to the table than what is visible.
