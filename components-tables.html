Tables

Add class="cb-table" to tables. This adds the styles that you see below. codebase.js adds a wrapper around tables that have this class, enabling responsive overflow (horizontal scrolling). If the table width is wider than the width allowed in the webpage layout, then a thick border is added to the right of the table to indicate that there is more to the table than what is visible.

A table that is horizontally scrollable on narrow screens:

Mixins
// COMPONENT MIXINS: TABLES
// ========================
// File: codebase/scss/07_component-mixins/_tables.scss

@mixin table {
  position: relative;
  border-collapse: collapse;
  min-width: 100%;
  margin-top: 0;
  margin-bottom: $base-line-height-rem;
	caption {
		padding: $deco-spacing-block;
	}
	th {
    text-align: left;
		background: rgba(204,204,204,0.2);
	}
	th, td {
    border: $deco-border-thin-solid;
		padding: $deco-spacing-sm;
    vertical-align: top;
	}
  &--wrap-outer {
    position: relative;
    z-index: 1;
    margin: auto;
    background: transparent;
  }
  &--wrap-inner {
    position: relative;
    overflow-x: auto;
    z-index: -1;
    margin: $deco-spacing-sm 0;
  }
  &--wide {
    box-shadow: $deco-box-shadow-inset;
  }
  &--nowrap {
    white-space: nowrap;
  }
}
