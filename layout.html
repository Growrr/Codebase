Media Query Breakpoints

The Codebase media query breakpoints are specified by three mixins. The breakpoint values can easiy be modified. 

$break-sm:  768px;
$break-md: 1024px;
$break-lg: 1200px;

Grid Systems

In Codebase three grid systems have been provided (Classic, Semantic and Flexbox). These all have the same characteristics:

  • 12 cols – you can change this simply by modifying the $grid-max-cols variable in your chosen grid flavour
  • Flexible – col width is set using max-width, not width
  • Responsive – media query breakpoint widths – you can change these in scss/base/_base-breakpoints.scss
  • Pushing/pulling (or in case of the flexbox grid, re-ordering)
  • Offsetting
  • Nesting
  • (single column) centering

These three grid systems all display essentially the same in browsers new enough to cope with flexbox (the only significant difference being that the flexbox cols are the full height of their containers, not merely the height of their own content).

Visibility at Different Breakpoints

In addition, visibility utilities (mixins and classes) have been provided, for showing or hiding items at different breakpoints.
