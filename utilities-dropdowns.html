Dropdowns can be added to buttons or menubar items (for creating child menus).

Examples:

Example dropdown utility content
 
Example dropdown utility content

Besides clicking the button again to dismiss the dropdown, you can click anywhere outside the dropdown to dismiss it (i.e. click on a transparent full-screen overlay, which is set at a z-index one unit less than the dropdown).

Use data-dropdown-content to point to the id of the dropdown__content that you wish to activate. As only data-dropdown-content will trigger the dropdown, this means that combo-sets can be put together (as above, example 2).

Use data-icon to specify the class(es) you wish to assign to the dropdown indicator icon. Here Font Awesome classes have been used. The Codebase dropowns jQuery script will read this and append the icon to the dropdown toggler (in these examples, the togglers are buttons).

The .dropdown wrapper ensures that the dropdown appears flush left of the button. The wrapper has display: inline-block so that you can easily build a row of dropdowns and other inline elements. 

<div class="dropdown">
  <button class="btn dropdown__toggle" data-dropdown-content="dropdown-1" data-icon="fa fa-caret-down">
    Example 1
  </button>
  <span id="dropdown-1" class="dropdown__content">
    Example dropdown utility content
  </span>
</div>
