Utilities:

Dropdowns

Dropdowns can be added to buttons or menubar items (for creating child menus).

Examples:

Example dropdown utility content
 
Example dropdown utility content


Besides clicking the button again to dismiss the dropdown, you can click anywhere outside the dropdown to dismiss it (i.e. click on a transparent full-screen overlay, which is set at a z-index one unit less than the dropdown).

Use data-dropdown-content to point to the id of the dropdown__content that you wish to activate. As only data-dropdown-content will trigger the dropdown, this means that combo-sets can be put together (as above, example 2).

Use data-icon to specify the class(es) you wish to assign to the dropdown indicator icon. Here Font Awesome classes have been used. The Codebase dropowns jQuery script will read this and append the icon to the dropdown toggler (in these examples, the togglers are buttons).

The .dropdown wrapper ensures that the dropdown content appears flush left of the button. The wrapper has display: inline-block so that you can easily build a row of dropdowns and other inline elements.

However, the downside of the .dropdown wrapper having display: inline-block is that the dropdown content is constrained to the width of the has .dropdown__toggle element. But there are various ways around this, which you can apply. One option is to add the modifier class .dropdown__content--nowrap (as used in example 1). Another option is for you to specify your required width for the dropdown content.

HTML
<div class="dropdown">
  <button class="btn dropdown__toggle" data-dropdown-content="dropdown-1" data-icon="fa fa-caret-down">
    Example 1
  </button>
  <span id="dropdown-1" class="dropdown__content dropdown__content--nowrap">
    Example dropdown utility content
  </span>
</div>

<div class="dropdown">
  <button class="btn combo--first">
    Example 2 (combo)
  </button>
  <button class="btn btn--icon combo--last dropdown__toggle" data-dropdown-content="dropdown-2" data-icon="fa fa-bars"></button>
  <span id="dropdown-2" class="dropdown__content">
    Example dropdown utility content
  </span>
</div>
Mixins
// UTILITY: DROPDOWNS
// ==================
// File: codebase/scss/06_utillity-mixins/_dropdowns.scss

@mixin dropdown {
  display: inline-block;
  position: relative;
  overflow: visible;
}

@mixin dropdown__toggle {
  &--is-toggled {
    position: relative;
    z-index: $reset-overlay--z-index + 1;
  }
}

@mixin dropdown__content {
  display: none;
  position: absolute;
  z-index: $reset-overlay--z-index + 2;
  margin: 0;
  padding: $deco-spacing-sm $deco-spacing-md;
  border-radius: $deco-border-radius;
  color: $base-text-color;
  background: #fff;
  text-align: left;
  cursor: default;
  box-shadow: $deco-box-shadow;
  &--is-revealed {
    display: block;
  }
  &--nowrap {
    white-space: nowrap;
  }
}
Classes
// UTILITY CLASS-BUILDER: DROPDOWNS
// ================================
// File: codebase/scss/10_utility-classes/_dropdowns.scss

.dropdown {
  @include dropdown;
}

.dropdown__toggle {
  @include dropdown__toggle;
}

.dropdown__content {
  @include dropdown__content;
}
