Incorporate the Codebase flexbox grid mixins into your own layout, using your own choice of class names.

The Codebase flexbox grid mixins are named similar to the Codebase classic grid classes, except that the col numerals are used as in the example SCSS stylesheet below (this arrangement of mixins would give you the example layout above).

You can also control the media query breakpoints using the Codebase breakpoint mixins, as below. 

.your-flexbox-grid-wrapper-class {
  @include grid-no-flexbox;
  @include grid--gutter;
  @include break__md--up {
    @include grid-flexbox;
  }
}

.your-flexbox-grid-article-class {
  @include col-flexbox;
  @include break__md--up {
    @include col-flexbox--span(8);
  }
  @include break__lg--up {
    @include col-flexbox--push(1);
  }
}

.your-flexbox-grid-sidebar-class {
  @include col-flexbox;
  @include break__md--up {
    @include col-flexbox--span(4);
  }
  @include break__lg--up {
    @include col-flexbox--pull(1);
  }
}

.your-flexbox-grid-nested-class {
  @include grid-no-flexbox;
  @include grid--gutter;
  @include grid--nested;
  @include break__sm--up {
    @include grid-flexbox;
    @include grid--nested;
    > div {
      @include col-flexbox;
      @include col-flexbox--span(4);
    }
  }
}

.your-flexbox-grid-centered-class {
  @include col-flexbox;
  @include break__sm--up {
    @include col-flexbox--centered(7);
  }
}
Mixin Explanation
@mixin grid-no-flexbox Container mixin, with display: block. Sets the max width (from variable $grid-max-width), does centering by auto margin right and left.
@mixin grid-flexbox Container mixin, with display: flex. Sets the max width (from variable $grid-max-width), does centering by auto margin right and left.
@mixin grid-flexbox--wrap Adds flex-wrap: wrap to the continer. For use if you have a surplus of columns at any time.
@mixin grid--gutter Gives right and left padding to all immediate child block elements (i.e. columns).
@mixin grid--nested Gives negative right and left margin to the grid container equal to the gutter width, so that a nested grid is flush right and left inside their containing column.
@mixin col-flexbox (Min height 1px.)
@mixin col-flexbox--centered(X) Centering is accomplished by auto margin right and left. Column spanning is as calculated out of $grid-max-cols (default is 12).
@mixin col-flexbox--span(X) Column spanning, as calculated out of $grid-max-cols (default is 12).
@mixin col-flexbox--push(X) Column pushing to the right using flexbox re-ordering. *--push(1) moves a column to the right by increasing its column order by 1.

E.g. pushing col [A] by 1 while pullng column [B] by 1 will result in the column order being swapped to [B][A].)
@mixin col-flexbox--pull(X) Column pulling to the left using flexbox re-ordering). *--pull(1) moves a column to the left by decreasing its column order by 1.
