Incorporate the Codebase flexbox grid mixins into your own layout, using your own choice of class names.

The Codebase flexbox grid mixins are named similar to the Codebase classic grid classes, except that the col numerals are used as in the example SCSS stylesheet below (this arrangement of mixins would give you the example layout above).

You can also control the media query breakpoints using the Codebase breakpoint mixins, as below.

        Mixin
        Explanation
      

    
    
      

        @mixin grid
        The container mixin. Sets the max width (from variable $grid-max-width), does centering by auto margin right and left.
          

          
If a containing wrapper doesn't have this grid mixin included, then a grid system will expand to whatever the width of the container is, without maxing out at $grid-max-width. E.g. it could expand to the BODY width.		
      

      

        @mixin grid--flexbox
        Container mixin, with display: flex. Sets the max width (from variable $grid-max-width), does centering by auto margin right and left.
      

      

        @mixin grid--flexbox--wrap
        Adds flex-wrap: wrap to the continer. For use if you have a surplus of columns at any time.
      

      

        @mixin grid--gutter
        Gives right and left padding to all immediate child block elements (i.e. columns).
      

      

        @mixin grid--nested
        Gives negative right and left margin to the grid container equal to the gutter width, so that a nested grid is flush right and left inside their containing column.
      

      

        @mixin col--flexbox
        (Min height 1px.)
      

      

        @mixin col--flexbox--centered(X)
        Centering is accomplished by auto margin right and left. Column spanning is as calculated out of $grid-max-cols (default is 12).
      

      

        @mixin col--flexbox--span(X)
        Column spanning, as calculated out of $grid-max-cols (default is 12).
      

      

        @mixin col--flexbox--push(X)
        Column pushing to the right using flexbox re-ordering. *--push(1) moves a column to the right by increasing its column order by 1. 

E.g. pushing col [A] by 1 while pullng column [B] by 1 will result in the column order being swapped to [B][A].)		
      

      

        @mixin col--flexbox--pull(X)
        Column pulling to the left using flexbox re-ordering). *--pull(1) moves a column to the left by decreasing its column order by 1.
