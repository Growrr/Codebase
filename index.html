Codebase

Codebase is a SASS, CoffeeScript, and Pug website component library.

Features include:

Two ways of using Codebase:

  • Simple setup
    – CSS, JS(jQuery) & HTML

  • Advanced setup
    – SASS, CoffeeScript & Pug

The styling has been created first as indented syntax SASS mixins and then these are included in classes – all following BEM naming convention. You can include and extend the mixins in your design theme.

The Styling is organised into Basics and Components (analogous to Atomic Design's "Atoms" and "Molecules"). For many of these, codebase-pug/ partials have been created for use in rapid prototyping (their usage is demonstrated in the docs Pug files). You can combine these into your own partial templates ("Organisms"), etc.

Layout is organized selarately from basics and components. Layout consists of four 12-column grid systems. Flexbox classes and mixins (default). Float classes and mixins (deprecated but still if you switch them on). A full range of visibility mixins/classes have also been included.

Minimal decorative styling used throughout the basics and components, based on a common set of decoration variables, which can be either modified or switched off altogether (becoming "style unoppinionated").

Simple decoration classes – similar in principle to BASCSS and Tachyons. These use the same decoration variables as Codebase Basics and Components, so they play well together, and modifying the decoration variables affects the basics, components and decoration.

All margins, paddings, font sizes and line heights are built on an 8px element grid.

Default and enlarged typography. Two sizes, three ways.

All Codebase variables are set as defaults, so that they can be easily overridden.

Extensive documentation, and a few examples.
