Utilities:

Buttons

Button classes can be applied to button tags, input-button tags and hyperlink anchor tags:

.btn

.btn

Hyperlink .btn

Three sizes (text stays the same size):

.btn .btn--sm

.btn

.btn .btn--lg

Block width button: .btn .btn--block

Button colors (note: the default is uncolored, i.e. transparent):

.btn

.btn .btn--primary

.btn .btn--secondary

.btn .btn--warning

.btn .btn--success

.btn .btn--error

The .btn--icon class can be used to make square buttons when only a (Font Awesome) icon is present within the button (note: .btn--icon cannot be used in conjunction with .btn--sm or .btn--lg).

.combo--* utility classes have the effect of removing some of the rounded forners of buttons (or form input boxes).

The dropdown utility can be used in buttons:

Example dropdown utility content
 
Example dropdown utility content

See the dropdown utility for explanation.

Example: icon buttons, combos and dropdowns (with added spacing, floats and clearfix wrappers to make everything appear in a line):

 
 
 
HTML
<!-- Button -->
<button class="btn">Button</button>

<!-- Input -->
<input class="btn" type="submit" value="Input">

<!-- Hyperlink -->
<a class="btn" href="">Hyperlink</a>

<!-- Button sizes -->
<button class="btn btn--sm">Small button</button>
<button class="btn">Default (medium) button</button>
<button class="btn btn--lg">Large button</button>

<!-- Block button-->
<button class="btn btn--block">Block width button</button>

<!-- Button colors -->
<button class="btn">Default</button>
<button class="btn btn--primary">Primary</button>
<button class="btn btn--secondary">Secondary</button>
<button class="btn btn--warning">Warning</button>
<button class="btn btn--success">Success</button>
<button class="btn btn--error">Error</button>

<!-- Square buttons for icons (default size only) -->
<button class="btn btn--icon"><i class="fa fa-arrow-right"></i></button>

<!-- Combos -->
<button class="btn combo--first">.combo--first</button>
<button class="btn combo--middle">.combo--middle</button>
<button class="btn combo--last">.combo--last</button>
Variables
// DEFAULT VARIABLES: BUTTONS
// ==========================
// File: codebase/scss/01_default-variables/_buttons.scss

$btn-rounded: true !default;
$btn-border-radius: $deco-border-radius !default;

$btn-padding-sm: $deco-spacing-xs $deco-spacing-sm !default;
$btn-padding-default: $deco-spacing-sm $deco-spacing-md !default;
$btn-padding-lg: $deco-spacing-md $deco-spacing-lg !default;
$btn-padding-icon: $deco-spacing-sm;

$btn-3d: true !default; // `false` gives flat buttons
$btn-pressable: true !default;
Mixins
// UTILITY MIXINS: BUTTONS
// =======================
// File: codebase/scss/06_utility-mixins/_buttons.scss

@mixin btn--3d {
  box-shadow:
    0 1px 1px rgba(0,0,0,0.5),
    inset 0 0 1px rgba(0,0,0,0.5),
    inset 0 10px 10px -10px rgb(255,255,255),
    inset 0 -20px 20px -20px rgba(0,0,0,0.4);
}

@mixin btn--pressable {
  &:active {
    box-shadow:
      0 0 0 transparent,
      inset 0 -50px 50px -50px rgba(0,0,0,0.4),
      inset 0 0 4px rgba(0,0,0,0.8);
  }
}

@mixin btn {
  display: inline-block;
  width: auto;
  margin: 0;
  border: 0;
  padding: $btn-padding-default;
  font-size: $base-font-size;
  line-height: $base-font-size;
  color: $base-text-color-muted;
  text-decoration: none;
  background: transparent;
  box-shadow: inset 0 0 1px rgba(0,0,0,0.2);
  &::-moz-focus-inner {
    border: 0;
    padding: 0;
  }
  &[type="button"],
  &[type="submit"] {
    -webkit-appearance: none;
  }
  &:hover {
    cursor: pointer;
    color: $base-text-color;
    text-decoration: none;
    background: $deco-silver;
  }
  @if $btn-rounded == true {
    border-radius: $btn-border-radius;
  }
  @if $btn-3d == true { // `false` gives flat buttons
    @include btn--3d;
  }
  @if $btn-pressable == true {
    @include btn--pressable;
  }
}

// Icon-only Buttons (default/md size only)

@mixin btn--icon {

  padding: $deco-spacing-sm 0;
  text-align: center;
