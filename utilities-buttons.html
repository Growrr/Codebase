Button classes can be applied to button tags, input-button tags and hyperlink anchor tags:

.btn

.btn (Note: input-buttons get margin-bottom: 10px from the form input tag styling.)

Hyperlink .btn

Three sizes (text stays the same size):

.btn .btn--sm

.btn

.btn .btn--lg

Block width button: .btn .btn--block

Button colors (note: the default is uncolored, i.e. transparent):

.btn

.btn .btn--primary

.btn .btn--secondary

.btn .btn--warning

.btn .btn--success

.btn .btn--error

The .btn--icon class can be used to make square buttons when only a (Font Awesome) icon is present within the button (note: .btn--icon cannot be used in conjunction with .btn--sm or .btn--lg).

.combo--* utility classes have the effect of removing some of the rounded forners of buttons (or form input boxes).

Example: icon buttons, combos and dropdowns (with added spacing, floats and clearfix wrappers to make everything appear in a line):

 
 
  
<!-- Button -->
<button class="btn">Button</button>

<!-- Input -->
<input class="btn" type="submit" value="Input">

<!-- Hyperlink -->
<a class="btn" href="#/">Hyperlink</a>

<!-- Button sizes -->
<button class="btn btn--sm">Small button</button>
<button class="btn">Default (medium) button</button>
<button class="btn btn--lg">Large button</button>

<!-- Block button-->
<button class="btn btn--block">Block width button</button>

<!-- Button colors -->
<button class="btn">Default</button>
<button class="btn btn--primary">Primary</button>
<button class="btn btn--secondary">Secondary</button>
<button class="btn btn--warning">Warning</button>
<button class="btn btn--success">Success</button>
<button class="btn btn--error">Error</button>

<!-- Square buttons for icons (default size only) -->
<button class="btn btn--icon"><i class="fa fa-arrow-right"></i></button>

<!-- Combos -->
<button class="btn combo--first">.combo--first</button>
<button class="btn combo--middle">.combo--middle</button>
<button class="btn combo--last">.combo--last</button>
