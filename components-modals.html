All modals in Codebase slide in from the top and appear flush with the top of the window (but you can move them in by adding a margin (or .deco-margin-top) to the modal body). The reset-overlay is raised and darkened, Modals can be dismissed by clicking their .modal__close button (or icon) or by clicking the reset-overlay.

The simplest modal is .modal--narrow with no modal header or footer:

Modal footer

If you require a modal to cover the entire window, add .modal--cover. For example, use this for a navigation system or sitemap. (Of course, full-cover modals can contain a grid, so there is no need to add .modal--contains-grid.)
